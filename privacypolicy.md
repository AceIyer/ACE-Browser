# Ace Site Blocker – Privacy Policy

**Last updated:** January 2026

Ace Site Blocker is a privacy-first browser extension designed to block access to unsafe and unwanted websites.

## Information Collection

Ace Site Blocker does **not** collect, store, transmit, or share any personal data or browsing information.

## Data Usage

All blocking rules and user preferences are processed and stored **locally on the user’s device** using Chrome’s local storage.  
No data is sent to external servers, third-party services, or analytics platforms.

## Permissions

The extension only uses browser permissions required to:

- Block access to specific websites  
- Detect navigation events  
- Apply user-configured blocking rules  

These permissions are used solely to provide the core functionality of the extension.

## Third-Party Services

Ace Site Blocker does **not** use any third-party services, trackers, advertisements, or analytics tools.

## Data Sharing

No user data is shared with any third party.

## Changes to This Policy

If the extension’s functionality changes in the future, this privacy policy will be updated accordingly.
